## MyTools - a handful of shell scripts to make your life easier.

### Installation

First, clone MyTools on your operating system.
Then run

```sh
cd ~
git clone https://github.com/armeninants/mytools.git
cd mytools
./myinstall
```

### How to use
To get started, run 
```sh
mydoc
```
or

```sh
mydoc <search_query>
```
